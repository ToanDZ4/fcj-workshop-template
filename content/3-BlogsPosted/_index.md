# Blogs Posted

During my internship I wrote three technical blog posts based on hands-on AWS labs I completed, and shared them with the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj) community. Each post walks through the goal of the lab, the architecture, the main steps, and the lessons learned.

### [Blog 1 – Hosting a static website on Amazon S3 and Amazon CloudFront](3.1-Blog1/)
A step-by-step lab on hosting a static website on Amazon S3 and delivering it globally over HTTPS with Amazon CloudFront, including bucket policies, Origin Access Control, and cache behavior.

### [Blog 2 – Building a serverless REST API with API Gateway, Lambda and DynamoDB](3.2-Blog2/)
How to build a fully serverless REST API where Amazon API Gateway routes requests to AWS Lambda functions that read and write to an Amazon DynamoDB table — no servers to manage and pay-per-request pricing.

### [Blog 3 – Running containers on Amazon ECS with AWS Fargate](3.3-Blog3/)
Deploying a containerized application on Amazon ECS with AWS Fargate behind an Application Load Balancer, covering the image push to Amazon ECR, task definitions, services, and health checks.
