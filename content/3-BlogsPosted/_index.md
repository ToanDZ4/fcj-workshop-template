# Blogs Posted

During my internship I wrote three technical blog posts about AWS services and shared them with the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj) community. Each post summarizes a service or feature, why it matters, how it works, and my own take on it.

### [Blog 1 – Cloud Security with Amazon VPC Encryption Controls](3.1-Blog1/)
An overview of Amazon VPC Encryption Controls, a new feature that lets organizations monitor, enforce, and audit encryption in transit between AWS resources — covering Monitor Mode vs. Enforce Mode, AWS Nitro hardware encryption, exclusions, and how it helps meet HIPAA, PCI DSS, and FedRAMP.

### [Blog 2 – Web Search on Amazon Bedrock AgentCore](3.2-Blog2/)
How the newly GA Web Search feature on Amazon Bedrock AgentCore lets AI agents look up current information on the web while keeping every query inside the customer's AWS environment (zero data egress), delivered as an MCP-standard connector with multi-source grounding and citations.

### [Blog 3 – Protecting Data on AWS with AWS Backup](3.3-Blog3/)
A practical look at AWS Backup as a fully managed, centralized backup and recovery service — Backup Plans, Backup Vaults, Cross-Region and Cross-Account backup, best practices, and why a backup strategy belongs in the architecture from day one.
