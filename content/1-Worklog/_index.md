# Worklog

This worklog documents my 12-week internship in the **First Cloud AI Journey (FCAJ)** program at **Amazon Web Services Vietnam**, from **17/04/2026** to **30/07/2026**.

Each week I followed the [AWS Cloud Journey](https://cloudjourney.awsstudygroup.com/) learning path, completed hands-on labs on a personal AWS Free Tier account, and progressively built a capstone project — a **Serverless E-commerce Platform** on AWS (see [Workshop](../5-Workshop/)). The first eight weeks focused on the core AWS service groups (compute, storage, networking, database, security, serverless, monitoring); the last four weeks were dedicated to designing, building, testing, and documenting the capstone.

The detailed log for each week is linked below.

**Week 1:** [Onboarding & AWS fundamentals (Console, CLI, EC2)](1.1-week1/)

**Week 2:** [Identity & security foundations with IAM](1.2-week2/)

**Week 3:** [Networking on AWS – VPC, subnets, security groups](1.3-week3/)

**Week 4:** [Storage & databases – S3, DynamoDB, RDS](1.4-week4/)

**Week 5:** [Compute & containers – EC2, ELB, Auto Scaling, ECS/Fargate](1.5-week5/)

**Week 6:** [Serverless – Lambda, API Gateway, SQS/SNS, EventBridge](1.6-week6/)

**Week 7:** [DNS, CDN & authentication – Route 53, ACM, CloudFront, Cognito](1.7-week7/)

**Week 8:** [Monitoring, logging & caching – CloudWatch, CloudTrail, ElastiCache](1.8-week8/)

**Week 9:** [Infrastructure as Code & capstone architecture design](1.9-week9/)

**Week 10:** [Capstone build (1) – IAM, S3 frontend, database](1.10-week10/)

**Week 11:** [Capstone build (2) – ECS backend, CloudFront, serverless](1.11-week11/)

**Week 12:** [Capstone finalize – monitoring, testing, cleanup & documentation](1.12-week12/)
