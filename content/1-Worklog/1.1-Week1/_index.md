* Understood what AWS is and the main service groups: compute, storage, networking, database and security.

* Became familiar with the AWS Management Console and how to navigate services.
* Installed and configured the AWS CLI (access key, secret key, default region, output format) and ran basic commands (`aws sts get-caller-identity`, `aws ec2 describe-regions`, `aws ec2 describe-instances`).
* Launched a first EC2 instance, connected over SSH, and attached an additional EBS volume.
* Learned to work in parallel across the Console and the CLI to manage AWS resources.
