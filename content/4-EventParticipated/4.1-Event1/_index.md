# Summary: First Cloud AI Journey – Kick-off & Cloud Fundamentals Workshop

### Event Objectives
- Introduce the First Cloud AI Journey (FCAJ) program, its goals and its 12-week roadmap.
- Give interns a solid foundation in cloud computing and core AWS concepts.
- Get hands-on with the AWS Management Console and AWS CLI.

### Main Content
- **What is cloud computing:** the shift from on-premises to the cloud, and its benefits (elasticity, pay-as-you-go, global reach).
- **AWS global infrastructure:** Regions, Availability Zones, and edge locations.
- **Core service groups:** compute (EC2), storage (S3), networking (VPC), and database.
- **The shared responsibility model** and account security basics (IAM, MFA).
- **Hands-on lab:** create an AWS Free Tier account, explore the Console, and run first AWS CLI commands.

### What I Gained
- A clear mental model of how AWS is organized and how the pieces fit together.
- Practical first steps: account setup, billing alerts, and CLI configuration.
- A clear picture of the 12-week learning path ahead and what the capstone project would require.
- Motivation from meeting the mentors and other interns in the program.
